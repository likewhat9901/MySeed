# npm 명령어 치트시트

> `frontend/` 폴더 안에서 실행

| 명령어 | 언제 쓰나 |
|---|---|
| `npm run dev` | 개발 서버 시작 → localhost:3000 |
| `npm run build` | 배포 전 최종 빌드 확인 |
| `npm run start` | 빌드 결과물 로컬 실행 |
| `npm run lint` | 코드 규칙 위반 검사 |
| `npm run gen:types` | DB 구조 바뀐 후 타입 재생성 |
| `npm install` | 패키지 전체 설치 (처음 세팅 or package.json 변경 후) |
| `npm install <이름>` | 새 패키지 추가 |
| `npm uninstall <이름>` | 패키지 제거 |
| `npm audit` | 보안 취약점 확인 |
